# Hand Gesture Recognition Using MediaPipe and TensorFlow

## Project Overview
This project is a real-time Hand Gesture Recognition System developed using Python, OpenCV, MediaPipe, TensorFlow, and Google Colab. It detects hand landmarks and recognizes different hand gestures using a trained deep learning model.

## Features
- Real-time hand detection
- Hand landmark extraction using MediaPipe
- Gesture classification using TensorFlow
- Works on images and webcam input

## Technologies Used
- Python
- Google Colab
- OpenCV
- MediaPipe
- TensorFlow
- NumPy
- Matplotlib
